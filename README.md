# Opero – Calculatrice Moderne pour iOS

**Opero** est une calculatrice élégante, rapide et intuitive pensée pour le quotidien.  
Interface minimaliste, historique intégré, conversions simples : tout est fait pour aller droit au but.

---

## Fonctionnalités

-  **Calculatrice complète** : opérations basiques et avancées.
-  **Historique intelligent** : garde une trace de tes calculs.
-  **Interface fluide et moderne** : animations propres et design iOS.
- ️ **Retour haptique** : ressent chaque interaction.
-  **Mode conversion** : bascule facilement vers les conversions de devises (si inclus).
- ️ **Dark Mode supporté**.

---

## Technologies utilisées

- **Swift / SwiftUI**
- **MVVM**
- **Haptics**
- **Animations SwiftUI**
- **API pour conversions**

## Politique de confidentialité

1. Introduction

Opero (« l’Application ») respecte votre vie privée.
Cette politique explique quelles données sont collectées, comment elles sont utilisées et quels services tiers y ont accès.

En utilisant Opero, vous acceptez les pratiques décrites ci-dessous.

2. Données collectées

2.1. Données collectées automatiquement

Certaines données non personnelles peuvent être collectées automatiquement, notamment :
	•	Informations sur l’appareil (modèle, version iOS).
	•	Performances de l’application.
	•	Interactions basiques avec l’interface (ex : fréquence d’utilisation).

Ces données servent uniquement à améliorer l’expérience utilisateur.

3. Publicités (Google AdMob)

Opero utilise Google AdMob pour afficher des publicités.
AdMob peut collecter :
	•	Identifiant publicitaire (IDFA),
	•	Informations générales sur l’appareil,
	•	Données d’utilisation anonymisées.

Google peut utiliser ces données pour personnaliser les publicités, si vous l’acceptez via le système iOS (App Tracking Transparency).

 Vous pouvez refuser le suivi directement dans les réglages iOS.
 Google est conforme au RGPD et CCPA.

Pour plus d’informations : https://policies.google.com/technologies/ads

4. Fonctionnalité IA (présente ou future)

Si vous utilisez une fonctionnalité basée sur l’intelligence artificielle
 (ex : génération de contenu, analyse, etc.), certaines données que vous choisissez
 d’envoyer pourront être transmises à un service externe (ex : API OpenAI ou équivalent).

Nature des données envoyées à l’IA
	•	Texte que vous envoyez volontairement (ex : requêtes, contenus).
	•	Données strictement nécessaires pour produire la réponse.

Ce que nous ne collectons PAS

 Aucun message privé n’est stocké sur nos serveurs.
 Aucun contenu n’est revendu ni utilisé à des fins publicitaires.
 Aucune donnée sensible n’est demandée.

Les fournisseurs d’IA peuvent analyser les requêtes pour améliorer leurs modèles, conformément à leurs politiques.
Tu peux préciser plus tard : “Nous utilisons OpenAI API avec l’option d’opt-out des données” si tu l’actives.

5. Pas de création de compte

Opero ne nécessite aucune inscription.
Nous ne collectons :
 aucune adresse mail,
 aucun numéro de téléphone,
 aucun identifiant personnel.

6. Partage de données

Nous pouvons partager certaines données uniquement avec des services tiers indispensables :
	•	Google AdMob (publicités)
	•	(Futur) Fournisseur API pour l’IA

Aucun autre partage n’est effectué.

7. Sécurité des données

Nous prenons des mesures pour protéger vos informations et minimiser les données collectées.
Cependant, aucun système n’est totalement sûr. L’utilisation de l’app implique l’acceptation de ce risque.

8. Droits de l'utilisateur (UE & monde)

Si vous êtes dans l’Union Européenne ou une région disposant de lois similaires, vous pouvez :
	•	Demander la suppression de vos données collectées automatiquement.
	•	Désactiver l’IDFA dans les réglages iOS.
	•	Refuser les publicités personnalisées.

9. Modifications de cette politique

Cette politique peut être mise à jour en cas d’ajout de fonctionnalités (ex : IA).
La date de mise à jour sera modifiée en conséquence.
